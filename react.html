Azura Banking React Website

I designed and created a React website for a fictional online banking app using styled components, the website is a basic template which allows it do adapt to any businesses needs, as you can easily customize the current sections or add new sections with just the click of a button. The website also utilizes the React Smooth scroll function, making navigating it a breeze. I also used React Hooks and React Router to create this project.
The Website is completely responsive and works just as well on small screen sizes as it does on large screens, as shown here in these screen shots.
